4
GitHub Personal Access Token
トークンの取得方法
1. GitHub → Settings → Fine-grained tokens を開く
2. Repository access: masatonishimura121-lab/yogibo-training を選択
3. Permissions → Contents: Read and write
4. 生成されたトークンをここに貼り付けて「保存」
処理中...
📖 使い方 ① Zoomから参加者CSVをエクスポート
Zoom → レポート → 使用状況 → ミーティング → 参加者レポートからエクスポート

② 講座の種類・開催日を選択
該当する研修種別ボタンを選び、開催日を入力してください

③ CSVをドラッグ＆ドロップ
アップロード後にマッチング結果のプレビューが表示されます

④ GitHubトークンを入力して「反映する」
初回のみトークンの設定が必要です。一度保存すれば次回から不要です

⑤ 数分後にダッシュボードが更新されます
GitHub Actionsの処理完了後（約1〜2分）、参加状況表・参加者詳細が自動更新されます
